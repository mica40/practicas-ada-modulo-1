Curso de desarrollo Frontend

Aprende a crear páginas web para empresas sin conocimientos previos, comienza a desarrollar sitios profesionales con tecnologías modernas. En este curso te enseñamos buenas prácticas, trucos, y cómo usar correctamente las tecnologías para poder crear un proyecto web.

  1. •Introducción a HTML.
  2. •Introducción a CSS.
  3. •Introducción a Sass.
  4. •Introducción a JavaScript.
  5. •Introducción a React.
